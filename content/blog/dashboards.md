Version 51 introduced a new concept in Opticloud: **custom flexible real-time dashboards**.

The concept was created to solve two problems:

1. Customers use the platform in very different ways across the organization.
2. Sharing data is often harder than it should be.

## Solving the First Problem

Opticloud is used by many kinds of roles:

- operators
- technicians
- SCADA programmers
- lean managers
- foremen
- executives

They all want better production performance, but they need different views of the same factory data.

## Creating a Dashboard

You can create as many dashboards as you need.

To create one:

1. Go to `Dashboard` in the home navigation after logging in.
2. Press `Add` in the top right corner.
3. Give the dashboard a name.
4. Add widgets for the machines and data points you want to track.
5. Drag, drop, resize, and arrange them as needed.
6. Press `Save`, then `View`.

## Why This Matters

A system like this makes it possible to build dashboards for different purposes very quickly.

Examples include:

- maintenance dashboards for technicians
- efficiency dashboards for OEE KPIs
- operator views for machine status
- management overviews for bottlenecks and throughput

The widgets can also change color depending on what is defined as critical, normal, or strong performance, making it easier to understand the production state at a glance.
